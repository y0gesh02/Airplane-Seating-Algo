# Airplane-Seating-Algo

Follow the steps below to set up and run the RSpec test:

## 1. Clone the repository

```bash
git clone https://github.com/y0gesh02/Airplane-Seating-Algo.git
cd airplane_seating_algo 
```

## 2. Install RSpec
```bash
gem install rspec
```

## 3. Run the test
```bash
rspec task_spec.rb
```
